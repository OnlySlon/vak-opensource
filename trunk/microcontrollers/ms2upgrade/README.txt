
--- THIS UTILITY IS NOT FUNCTIONAL YET! ---

This directory contains a sources of mk2upgrade - utility for upgrading
a firmware of Microchip Microstick II board.

=== Usage ===

When called without parameters, mk2upgrade utility detects a board
and prints a version of firmware.  For example:

    % mk2upgrade

Writing to flash memory:

    mk2upgrade file.hex
